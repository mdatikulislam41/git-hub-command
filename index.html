How to work with any git commit(version)
  • You have to find Commit Id by bello command
  • git log --oneline
  • Then you have to copy commit Id(17ada02)
  • git checkout 17ada02 -- .
  • Now you can work this commit
How to reset(remove) git commit
  • There are 2 place for reset
  • 1. in repository 2. Vs-Code
  • but have to do this by command
  • First reset for Repository commit
  • You have to find Commit Id by bello command
  • git log --oneline
  • Then you have to copy commit Id(17ada02)
  • git reset --soft 17ada02
  • git push --force
  • Second reset for Vs code commit
  • git reset --soft 17ada02
  • git reset --hard 17ada02
  • Note: commit will remove which one top of 17ada02

Merge Branch
  • Merge মানে opening(active) branch এর সাথে অন্য একটি branch কাজ একত্রিত করা।
  • নোট : Merge করার পরে add, commit এবং push করতে হবে।
  • নিচের command দিয়ে merge করি
  • git merge [branch]
  • git add .
  • git commit -m "message"
  • git push
